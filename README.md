# Dashboard Interaktif - Tembaga: Jantung Elektrifikasi Nasional

Proyek ini adalah versi interaktif dan dinamis dari infografis statis mengenai "Optimalisasi Hilirisasi Tembaga dalam Mendukung Transisi Energi dan Pembangunan Hijau". Aplikasi berbasis web ini memungkinkan pengguna untuk menyaring, mengklik, dan mengeksplorasi data hilirisasi tembaga secara mandiri.

## 🚀 Fitur Utama Dashboard
1. Menyaring Data (Dynamic Filtering): Pengguna dapat menyaring rentang tahun projeksi pertumbuhan permintaan tembaga pada grafik interaktif (Recharts).
2. Mengklik & Hover (Interactive Components): Informasi padat seperti "Smart Smelting" dan "Pusat Teknologi AI" dikemas dalam bentuk tab interaktif dan KPI cards untuk menjaga UI tetap bersih.
3. Eksplorasi Mandiri (Simulator Kustom): Simulator berbasis React State yang menghitung otomatis total kebutuhan tembaga berdasarkan input jumlah produksi kendaraan EV vs ICE secara real-time.

## 🛠️ Spesifikasi Teknologi (Tech Stack)
- Frontend Framework: React.js (Vite)
- Styling: Tailwind CSS
- Data Visualization: Recharts Library
- Icons: Lucide React

## 💻 Cara Menjalankan Proyek Secara Lokal

Jika Anda mengunduh source code proyek ini dan ingin menjalankannya di komputer lokal, ikuti langkah-langkah berikut:

1. Pastikan Anda sudah menginstal Node.js di komputer Anda.
2. Ekstrak file ZIP proyek ini dan buka foldernya melalui Terminal / Command Prompt.
3. Instal semua dependencies yang dibutuhkan dengan mengetik perintah:
   npm install

4. Jalankan server pengembangan lokal dengan perintah:
   npm run dev

5. Buka browser dan akses tautan lokal yang muncul di terminal (biasanya http://localhost:5173).
